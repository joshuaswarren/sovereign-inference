# router-demo (scaffolded)

An end-to-end demo: start a local SIN provider, route a request through the
SIP-AI router, show the quote/payment/response and the signed receipt, then kill
the provider and show failover to another provider.

**Status:** scaffolded — this is the centerpiece of the
[demo script](../../docs/mvp-and-demo.md) and the hackathon
[evidence plan](../../docs/hackathon/evidence-plan.md). Built in Phase 2 of the
[ROADMAP](../../ROADMAP.md).
