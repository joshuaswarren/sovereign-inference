# Dashboard (scaffolded)

The local web dashboard for a Sovereign Inference Node: health, the running
model, benchmark results, sharing status, requests, receipts, and pause/resume.

**Status:** scaffolded — implementation is Phase 1 in the [ROADMAP](../../ROADMAP.md).
Planned stack: React + Vite (TypeScript), later wrapped by a Tauri desktop shell
that bundles the Python node as a sidecar.
